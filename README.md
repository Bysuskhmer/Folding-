# Library 

Library Editer By @bysuskhmer 

## Getting Loadstring

```lua
local Library = loadstring(game:HttpGet("https://github.com/UhGbaaaa/X-Mobile-/raw/main/Kavo%20V2.txt"))()
```

## Creating UI Library Window

```lua
local Window = Library.CreateLib("TITLE", "DarkTheme")
```
